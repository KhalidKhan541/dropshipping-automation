# Dropshipping Automation

Automate your dropshipping business with AI-powered tools. Product research, order processing, and analytics.

## Features
- Product research automation
- Order processing
- Analytics dashboard
- TikTok scheduler
- WhatsApp auto-reply

## Tech Stack
- TypeScript
- Cloudflare Workers
- Cloudflare Pages

## Run Locally
```bash
npm install
npm run dev
```

## Deploy
```bash
npm run build
npx wrangler pages deploy dist
```
